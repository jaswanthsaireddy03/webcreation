control structures decision making and branching

In C language, decision-making and branching control structures are used to make decisions based on certain conditions and execute different sets of statements based on those decisions.
The if statement is a common decision-making control structure in C. It allows the program to execute a block of code if a certain condition is true. Here's an example:
int num = 5;
if (num > 0) {
printf("The number is positive.\n");
}
In this example, the if statement checks whether the value of num is greater than 0. If it is, the program executes the block of code inside the curly braces { }, which in this case, prints a message to the console indicating that the number is positive.
The if-else statement is another decision-making control structure in C. It allows the program to execute one block of code if a certain condition is true, and another block of code if the condition is false. Here's an example:
int num = -2;
if (num > 0) {
printf("The number is positive.\n");
} else {
printf("The number is non-positive.\n");
}
In this example, the if-else statement checks whether the value of num is greater than 0. If it is, the program executes the first block of code, which prints a message indicating that the number is positive. If it is not, the program executes the second block of code, which prints a message indicating that the number is non-positive.
The switch statement is another decision-making control structure in C. It allows the program to execute different blocks of code based on the value of an expression. Here's an example:
int num = 3;
switch (num) {
case 1:
printf("The number is one.\n");
break;
case 2:
printf("The number is two.\n");
break;
case 3:
printf("The number is three.\n");
break;
default:
printf("The number is not one, two, or three.\n");
break;
}
In this example, the switch statement checks the value of the variable num. Depending on the value, the program executes a different block of code. If num is 1, the program prints a message indicating that the number is one. If num is 2, the program prints a message indicating that the number is two. If num is 3, the program prints a message indicating that the number is three. If num is any other value, the program executes the default block of code, which prints a message indicating that the number is not one, two, or three.
Overall, decision-making and branching control structures are important for creating dynamic and flexible programs that can handle different situations based on certain conditions.
