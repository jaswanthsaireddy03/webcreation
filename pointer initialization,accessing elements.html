In C programming language, a pointer to pointer is a variable that holds the memory address of another pointer. A pointer to structure is a variable that holds the memory address of a structure.
Here is an example program that demonstrates pointer to pointer and pointer to structure:
#include
struct student {
char name[50];
int age;
};
int main() {
struct student stu = {"John Doe", 20};
struct student *ptr1;
struct student **ptr2;
ptr1 = &stu;
ptr2 = &ptr1;
printf("Name: %s\n", (*ptr2)->name);
printf("Age: %d\n", (*ptr2)->age);
return 0;
}
In this program, a structure student is defined with two fields name and age. Then, an instance of the structure stu is created and initialized with the values "John Doe" and 20.
Next, a pointer ptr1 to the structure student is declared and initialized to the memory address of stu. Then, a pointer ptr2 to a pointer to the structure student is declared and initialized to the memory address of ptr1.
To access the fields of the structure stu using the pointer to pointer ptr2, we use the -> operator twice: the first -> operator dereferences ptr2 to obtain ptr1, and the second -> operator dereferences ptr1 to obtain the structure stu.
The program then prints the name and age of the structure stu using the dereferenced pointer to pointer ptr2.
In C programming language, a pointer is a variable that holds the memory address of another variable. Pointers can be initialized to a memory address using the & operator, which returns the address of a variable, or using a pointer constant, which is a memory address written in hexadecimal notation.
Here is an example program that demonstrates pointer initialization and accessing elements using pointer arithmetic:
#include
int main() {
int arr[5] = {1, 2, 3, 4, 5};
int *p, i;
p = &arr[0];
for (i = 0; i < 5; i++) {
printf("*(p+%d) = %d\n", i, *(p+i));
}
return 0;
}
-->In this program, an integer array arr of size 5 is declared and initialized with the values 1, 2, 3, 4, and 5. Then, a pointer p to an integer is declared.
The pointer p is initialized to the memory address of the first element of the array arr using the & operator.
To access the elements of the array arr using the pointer p, we use pointer arithmetic. The expression *(p+i) returns the value at the memory location that is i integers away from the memory location pointed to by p. The printf statement in the for loop prints the values of the array arr using the pointer p and pointer arithmetic.
Note that pointer arithmetic can also be used to modify the values of the array elements using pointers. For example, the statement *(p+2) = 10 sets the value of the third element of the array arr to 10.
